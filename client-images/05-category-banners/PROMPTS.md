# 05 — Category Banners (top of each category page)

6 wide banners. Each needs **Desktop (1600×520, wide)** + **Mobile (1000×700)**.
Prepend the **GLOBAL STYLE**. These sit behind the category title, so keep the **left side open**
(desktop) / **top open** (mobile) for the heading, and a soft dark gradient is added automatically.
*Optional but recommended — gives each category page a rich, branded hero.*

> Reusable composition note: **[MY CLIENT]** on one side using/holding that category's gear, the rest
> open negative space, warm‑dark premium tones.
